Welcome to the Prompt Evaluation System

For each contribution, new features, improvement etc use a top-down approach. Start with a general idea, and detail it over time. That's why if a contribution lowers one or more scores that's totally fine — especially for new changes. If you believe that even after improvements, a new edit still lowers overall potential compared to the previous version, just bring it up in the group. We’ll discuss and decide it together.

Write your prompts in the editor below. Use "===" to separate different prompts. (exemple "build a todo app with an apple design" is one prompt". "Build a todo app === Make it apple like" count as a chain of 2 prompt). Also try to be coherent with the order of prompts.

Accuracy. Evaluates how precise the prompt is by testing it across multiple AI models and comparing the consistency of their outputs.

Complexity. Assesses the scope of the project described in the prompt, based on how challenging it would be to build.

Reliability. Measures how well the prompt anticipates and addresses edge cases, errors, vulnerabilities and major security flaws.

Pro tip: To increase the scores, Be specific and include code snippets to increase accuracy, add or detail features to increase complexity, and consider edge cases and error for better reliabilty.
